# ICAR-4389 开发任务清单版（前端主导）

## 1. 目标与交付原则

- 目标：将「佣金汇率结算配置」PRD落地为可执行开发任务，支持排期、分工、联调和验收。
- 原则：先 P0 主链路上线（配置 + 统计 + Admin兼容），再做 P1 优化。
- 输出粒度：每项任务包含负责人角色、前置依赖、完成定义（DoD）。

---

## 2. 里程碑与阶段划分

## M1（P0）必须首发
- OWS 菜单与权限
- 汇率配置页（含编辑、保存、2FA、历史记录）
- 汇率统计页（利润统计 + 利润明细）
- Admin 报表兼容（开仓点差、平仓价）
- 国际化文案（中英）

## M2（P1）增强优化
- 报表体验优化（复杂表格交互）
- 监控与埋点增强
- 性能与弱网体验优化

## 4. 前后端联调清单

## 4.1 接口与字段联调
- 配置读取/保存：X/Y/Z 的空值语义、范围语义、精度语义统一。
- 历史记录：From/To 字段是否服务端直接返回格式化文本。
- 统计报表：行列口径、Total 计算边界、USD换算精度。
- 明细列表：总利润字段是否后端返回，或前端按行汇总。
- Admin兼容：扣点后开仓点差、平仓价字段命名与回填时机。

## 4.2 错误码与提示映射
- 2FA 失败、过期、取消
- 输入超限、非法精度
- 无权限、数据为空、系统异常

## 4.3 时区与时间
- 页面统一时区口径
- 日期边界（00:00:00 / 23:59:59）与后端一致

## 5. 测试清单（可直接执行）

## 5.1 功能测试
- 三项汇率输入：空值、边界值、非法值、两位小数限制。
- 品牌切换：配置独立、历史独立、筛选状态是否保留。
- 保存流程：2FA 成功/失败/超时三路径。
- 统计页：行列顺序、Total 逻辑、默认日期范围。
- 明细页：监管单选、日期上限30天、Total Profit 计算。
- Admin页：开仓点差与平仓价展示值正确。

## 5.2 权限测试
- 无菜单权限：菜单不显示。
- 有菜单无编辑权限：可看不可改。
- URL 越权：拦截并提示。

## 5.3 回归测试
- IB佣金、销售佣金、PAMM返佣、跟单返佣
- IB分佣、佣金监控、佣金重算、API同步
- IB portal / Davinci / SWS / Admin 相关展示

## 7. 交付完成定义（Definition of Done）

- 功能：P0 功能全部可用，且与 PRD 关键条目逐项对齐。
- 质量：单测/E2E/回归通过，关键异常路径可复现可恢复。
- 安全：权限与2FA流程闭环，无越权入口。
- 可观测：埋点与监控生效，可追踪保存成功率和报错率。
- 发布：完成小品牌灰度观察并具备一键回滚能力。
